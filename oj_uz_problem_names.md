## Бодлого Хайх Аргууд

### Alias-аар хайх
Бодлогын alias код нь:
- **APIO** + он + "_" + нэр: APIO19_bridges
- **IOI** + он + "_" + нэр: IOI18_combo
- **BOI** + он + "_" + нэр: BOI19_nautilus
- **JOI** + он + "_" + нэр: JOI21_navigation2
- **NOI** + он + "_" + нэр: NOI19_pilot

### URL-аар шууд очих
```
https://oj.uz/problem/view/{ALIAS}
```
